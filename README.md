# 🏔️ Nature Tours — Hero Page

A full-screen hero landing page built as my first Tailwind CSS project.
Focused on layout, typography, animations, and responsive design.

## 🌿 Preview

landing.png

## ✨ Features

- Full-screen hero section with mountain background image
- Glassmorphism overlay effect with `backdrop-blur`
- Animated heading and bounce icon
- Responsive centered layout using Flexbox
- Two CTA buttons with hover transitions
- Built with Vite for fast development

## 🛠️ Built With

- HTML
- Tailwind CSS
- Vite

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 💡 What I Learned

This was my first hands-on project with Tailwind CSS.
I practiced utility classes, responsive design, animations,
and how to set up a project with Vite.
